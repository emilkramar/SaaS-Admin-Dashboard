# Screenshots for the README

These images are referenced from the root [`README.md`](../../README.md).

Replace them with **real captures** from your machine so the repo matches your exact UI:

1. Run `npm start` and open `http://localhost:4200`.
2. Capture:
   - Dashboard (light)
   - Users page with the table toolbar
   - Dashboard or users in **dark mode** (toggle in the header)
3. Export as PNG and keep the filenames:
   - `dashboard-overview.png`
   - `users-table.png`
   - `dashboard-dark.png`

Alternatively, rename your files and update the image paths in the main README.
